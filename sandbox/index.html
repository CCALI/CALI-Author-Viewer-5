As I said before, "market" usually means replacement cost, but there are two exceptions:

(1) If replacement cost is greater than net realizable value, you use net realizable value as the "market" value instead.

(2) If replacement cost is less than net realizable value less normal profit, you use net realizable value less normal profit as the "market" value instead.

To put it another way, "market" for purposes of the rule can never exceed net realizable value and can never be less than net realizable value less normal profit.
