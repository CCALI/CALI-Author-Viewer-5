CALI Lessons Choose Multi Buttons

These are the choose multi buttons for when students can select from multiple answers. There may be more than one correct answer and some answers might also be maybe.

(NOTE for Dutch: The hints on the buttons are set for english text sizing. To give more spacing for hint text you will need to change the css in .btn-group and change margin-right: 0px; to the needed pixel margin for spacing to the right.)
